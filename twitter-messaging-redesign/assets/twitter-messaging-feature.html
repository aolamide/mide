The Twitter messaging feature

Currently, this is how the Twitter messaging feature looks like;

Twitter messaging screen: Where users can see their recently contacted, their message requests and have the option to start a new chat.

Message requests screen: This is the screen where spam/unknown messages come in. It is automatically filtered out from the important messages.

The Messages settings screen: this is where users customize their messages however they want. Be it privacy, read receipts and push notifications.
